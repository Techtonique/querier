The `querier` is a query language for data frames. Its **source code** is `available on GitHub <https://github.com/thierrymoudiki/querier>`_, and you can read blog posts about the tool `here <https://thierrymoudiki.github.io/blog/index.html#DataBases>`_. There is also a repository with loads of examples `on GitHub <https://github.com/thierrymoudiki/querier/tree/master/querier/demo>`_.
